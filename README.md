pkprint
========

# Description
print pickle object

# Usage

```bash
$ pkprint python pkprint.py -h
usage: pkprint.py [-h] path

print pickle object

positional arguments:
  path        path for pickle

optional arguments:
  -h, --help  show this help message and exit

```

# Example

```bash
python3 pkprint.py sample.pickle
```
